# Python → C++ 工业视觉教程：设计文档

## 目标

为 Python 基础好、C++ 零基础的开发者，生成一份"可运行、可阅读、可理解"的详细教程，包含：

1. **TUTORIAL.md** — 独立教程文档，逐模块讲解 C++ 概念（对照 Python）
2. **源码注释增强** — 在现有 `.cpp` / `.hpp` 中补充中文对照注释

## 范围

| 做 | 不做 |
|---|------|
| 覆盖 00 ~ 07 全部 8 个模块 | 不修改代码逻辑和功能 |
| C++ 概念与 Python 语法对照 | 不添加新功能或新模块 |
| 工业视觉背景说明 | 不重写 CMakeLists.txt |
| 运行结果解读 | 不修改项目结构 |
| 安装 ONNX Runtime / torch / pybind11 并验证 | |
| 源码添加中文概念注释和 Python 对照注释 | |

## 交付物

### 1. TUTORIAL.md（约 15000-20000 字）

每章结构：

```
# 第 N 章：标题

## 1. 问题背景
工业视觉中这个环节解决什么问题

## 2. 概念速览
C++ 新概念 | Python 对照
新概念列表，含 table

## 3. 代码逐行解析
三段式：C++ 代码片段 → Python 对照 → 解析

## 4. 运行结果解读
运行输出 + 分析 + 思考题
```

### 2. 源码注释

在每个 `.cpp` 和 `.hpp` 文件中追加：

- `// --- 概念: xxx ---` 解释 C++ 语法
- `// Python: xxx` 标注对应 Python 写法

注释密度分三级：
- 首章（第 1 章/模块00、第 2 章/模块01）：每 3-5 行
- 中间（第 3/5/6 章/模块02/04/05）：每 8-10 行
- 后续（第 4/7/8 章/模块03/06/07）：关键处标注

## 技术约束

- C++17 标准
- 面向 Python 3 基础扎实的读者（教程本身使用 Python 3.14）
- 所有示例在纯 CPU Linux 可运行
- ONNX Runtime 1.26.0（最新稳定版）
- OpenCV 4.10
- pybind11 2.13.6

## 验收标准

1. `TUTORIAL.md` 每章包含：背景、概念表、代码解析、运行解读四个小节
2. 所有源码文件追加了中英文对照注释
3. 每章末尾至少有 1 个思考题
4. 所有模块编译通过、运行通过
5. Python 开发者阅读后能理解 C++ 对应写法
